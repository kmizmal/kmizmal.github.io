Quick Start

Create a new post

$ hexo new "My New Post"

More info: Writing

Run server

$ hexo server

More info: Server

Generate static files

$ hexo generate

More info: Generating

Deploy to remote sites

$ hexo deploy

$ hexo deploy
$hexo new "name"       # 新建文章
$hexo new page "name"  # 新建页面
$hexo g                # 生成页面
$hexo d                # 部署
$hexo s                # 本地预览
$hexo clean            # 清除缓存和已生成的静态文件
$hexo help             # 帮助
$hexo g -d             # 生成页面并部署
